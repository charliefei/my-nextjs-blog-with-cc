## 项目概述

使用 Next.js 15 构建的功能丰富的个人博客，包含以下特性：

- **国际化**: 使用 next-intl 实现完整的中英文支持
- **主题系统**: 深色/浅色模式，支持系统偏好检测
- **内容管理**: 基于 Markdown 的文章，使用 gray-matter 解析 frontmatter
- **静态导出**: 可部署到 Vercel 或 GitHub Pages

## 技术亮点

- 使用 Next.js App Router 获得最佳性能
- TypeScript 提供类型安全和更好的开发体验
- Tailwind CSS 配合 shadcn/ui 实现美观、响应式的设计
- 静态站点生成，加载快速且 SEO 友好

## 功能特性

- 博客文章支持标签、分类和搜索
- 简历页面支持 PDF 查看器
- 工作经历时间线和技能可视化
- RSS 订阅生成
- SEO 优化，支持 Open Graph
