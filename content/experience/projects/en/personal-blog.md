## Project Overview

A feature-rich personal blog built with Next.js 15, featuring:

- **Internationalization**: Full support for English and Chinese with next-intl
- **Theme System**: Dark/light mode with system preference detection
- **Content Management**: Markdown-based posts with gray-matter frontmatter parsing
- **Static Export**: Deployable to Vercel or GitHub Pages

## Technical Highlights

- Built with Next.js App Router for optimal performance
- TypeScript for type safety and better developer experience
- Tailwind CSS with shadcn/ui for beautiful, responsive design
- Static site generation for fast loading and SEO optimization

## Features

- Blog posts with tags, categories, and search
- Resume page with PDF viewer
- Experience timeline with skills visualization
- RSS feed generation
- SEO optimized with Open Graph support
